Blog Sidebar Widget provides you necessary widgets for better and effective blogging.

== Description ==

Blog Sidebar Widget is a lightweight and safe plugin that generates 9 much-necessary custom widgets. This plugin is optimized for avidthemes themes but also works great with other themes. 

= Widgets included in  Blog Sidebar Widget = 

* Blog Sidebar: Advertisement-- This widget lets you add advertisements on your website.
* Blog Sidebar: Author Bio-- You can place the author bio on sidebars using this widget.
* Blog Sidebar: Contact Widget-- You can place the contact info using this widget.
* Blog Sidebar: Custom Categories-- You can add images to the categories and also display post counts of the selected category using this widget.
* Blog Sidebar: Featured Page-- This widget allows you to a specific page inro in your WordPress website.
* Blog Sidebar: Image Text-- You can add your favorite image with caption and link with this widget.
* Blog Sidebar: Popular Post- This widget helps you show popular posts of your website.
* Blog Sidebar: Posts Category Slider-- You can display the posts of selected category in a slider using this widget.
* Blog Sidebar: Recent Post-- This widget helps you display recent posts of your website.



== Installation ==
**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search for 'Blog Sidebar Widget' and install the plugin.
3. Activate ‘Blog Sidebar Widget’ from your Plugins page.
